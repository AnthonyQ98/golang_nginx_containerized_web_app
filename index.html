Who Am I

I am a Security DevOps Engineer at Salesforce, and slightly addicted to programming and Brazilian Jiu Jitsu.

More about me

I've worked with large scale systems and infrastructure for almost three years now, in production and development environments. I've worked within the public cloud (AWS, mostly, but also used OpenStack and some others!) and on-premise Salesforce-owned infrastructure.

Implementing security controls and policy is my bread and butter. I've worked extensively with OPA and applied it to clusters in EKS to enforce best practices organization-wide.

I've also developed wrappers for Linux package management tools, such as Yum (or DNF in RedHat), to remove insecure packages from our systems that didn't follow proper change control policy within a specified timeframe.

Recently, I developed tooling for GitHub using the GitHub GraphQL and REST APIs to audit and evaluate config settings on organizations and repositories within GitHub Enterprise Cloud (GHEC).

Extra projects on my personal GitHub is simply an opportunity to further expand my developer skills, using the latest industry standard practices & methods. Admittedly, I don't find much time for this outside of my work hours nowadays, but I try to work on fun, new stuff, when I can!
